# AI Sales Buddy
## Implementation Plan & Commercial Proposal

**Prepared for:** Hyundai Motor India  
**Prepared by:** Maigic.AI  
**Version:** 1.0 | December 2025

---

<br/>

## Executive Summary

We propose to build the AI Sales Buddy in **two phases over 7 months**, leveraging our existing VahanX AI platform to accelerate development and minimize risk.

| | Phase 1 | Phase 2 |
|---|---------|---------|
| **Duration** | 4 months | 3 months |
| **Scope** | Core platform + 2 focus models | Full model range + advanced features |
| **Go-Live** | Pilot with select dealers | National rollout |

**Total Investment:** ₹50,00,000 (Fifty Lakhs)

<br/>

<br/>

## Phase 1: Foundation & Pilot (4 Months)

### Scope - What Gets Delivered

| Category | Features |
|----------|----------|
| **Core Platform** | AI chatbot with text input/output, iFrame widget, SSO integration |
| **Product Coverage** | 2 focus models (e.g., Creta, Venue) - complete specs, features, variants |
| **Competitor Comparison** | Comparison with key competitors for 2 models |
| **Sales Process** | Standardized sales process guidance from LMS |
| **Languages** | English + Hindi (2 languages) |
| **Voice Input/Output** | Speech-to-text, text-to-speech for English & Hindi |
| **Analytics** | Basic dashboard - sessions, queries, active users |
| **Integration** | LMS/H-Smart App iFrame, Hyundai SSO |
