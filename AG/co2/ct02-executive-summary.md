# CloudTuner CO2 Token (CT02) - Executive Summary & Quick Reference

## Your CO2 Token at a Glance

### What is CT02?

**CT02 is a blockchain-based token where 1 token = 1 kg of CO2 emissions from cloud computing (EC2, Azure VMs, GCP compute).**

It's NOT a fixed-supply token. Instead, the supply grows dynamically as your customers consume cloud resources and generate carbon emissions.

## The Core Model in 60 Seconds

### Step 1: Customer Uses Cloud
- ABC Corp runs 100 EC2 instances on AWS
- CloudTuner tracks carbon: **50 kg CO2 this month**

### Step 2: CloudTuner Mints Tokens
- 50 kg CO2 = 50 CT02 tokens automatically created
- Distribution (70/20/10 model):
  - **35 CT02** → ABC Corp (70%)
  - **10 CT02** → CloudTuner Treasury (20%)
  - **5 CT02** → Staking Reward Pool (10%)

### Step 3: Customer Chooses (3 Paths)

**Path A: OFFSET (Environmental)**
- ABC Corp retires 35 CT02 tokens
- Tokens permanently burned (destroyed)
- Gets digital certificate: "35 kg CO2 offset via CloudTuner"
- Can market: "Our cloud infrastructure is carbon-neutral"

**Path B: STAKE (Financial)**
- ABC Corp stakes 35 CT02 for 365 days
- Earns 50% APY = 17.5 CT02/year bonus
- After 1 year: 35 + 17.5 = 52.5 CT02 (50% gain)
- No environmental benefit, but financial reward

**Path C: TRADE (Liquidity)**
- ABC Corp sells 35 CT02 on marketplace
- Price: $50/token (example)
- Gets $1,750 USD instantly
- Uses for other projects

## The Deflationary Magic (Why Price Appreciates)

### How Tokens Disappear (Get Burned)

1. **Customer Retires Tokens** (Primary)
   - Customer burns CT02 for offset certificate
   - Example: 35 CT02 burned = 35 kg CO2 permanently offset
   - Token supply permanently shrinks

2. **Transaction Fees** (Secondary)
   - Every token trade: 1% of value automatically burned
   - Example: $1,000 trade at $50/token = ~100 CT02 burned

3. **Staking Reward Tax**
   - Every reward claimed: 1% burned
   - Example: Earned 50 CT02 reward → 0.5 CT02 burned

### Year 1 Burn Scenario
```
Minted: 35,500 CT02
Burned: 3,000 CT02 (8.5% of supply)
Net Supply: 32,500 CT02

Result: 
- Fewer tokens in circulation
- Higher scarcity
- Potential price appreciation
- Early holders benefit most
```
